# 外部平台发布模板和工具

## 🎯 使用说明

每个平台都有其独特的文化和格式要求。这些模板经过优化，可以帮助你在各平台获得最佳效果。

## 📝 Medium发布模板

### 文章结构模板

```markdown
# [吸引人的标题 - 使用数字或强烈词汇]

[150-200字的引人入胜的导语，设置悬念或提出问题]

## [小标题1: 清晰的价值主张]

[内容段落，包含个人经验、数据支撑]

> [引用：专家观点或关键洞察]

## [小标题2: 实用建议]

- [具体步骤1]
- [具体步骤2]
- [具体步骤3]

## [小标题3: 真实案例]

[详细的案例研究，包含背景、行动、结果]

## 结论

[2-3句话总结，包含行动号召]

---

**喜欢这篇文章吗？**
- 👏 点击鼓掌按钮（最多50次）
- 💬 在评论区分享你的想法
- 📬 订阅我的Medium专栏
- 🔗 [原文链接](canonical URL)
```

### 标题优化模板

**数字列表型**:
- "7种方法..."
- "5个步骤..."
- "3个关键要素..."

**对比型**:
- "为什么X比Y更好..."
- "A和B的真正区别..."

**承诺型**:
- "如何用30天..."
- "掌握这个技能的完整指南..."

**警告型**:
- "停止这样做..."
- "你可能一直在错..."

- [ ] 标题包含主要关键词

- [ ] 使用5-7个相关标签

## 🤖 自动化工具脚本

### 多平台发布脚本（Python示例）

```python
import requests
import json
from pathlib import Path

class ContentPublisher:
    def __init__(self, config_file='config.json'):
        with open(config_file) as f:
            self.config = json.load(f)

    def publish_to_medium(self, article_data):
        """发布到Medium"""
        headers = {
            'Authorization': f'Bearer {self.config["medium_token"]}',
            'Content-Type': 'application/json',
        }

        # 创建Medium文章
        url = 'https://api.medium.com/v1/users/'
        user_id = self.get_medium_user_id(headers)
        url += f'{user_id}/posts'

        payload = {
            'title': article_data['title'],
            'contentFormat': 'markdown',
            'content': article_data['content'],
            'tags': article_data.get('tags', []),
            'publishStatus': 'public'
        }

        response = requests.post(url, headers=headers, json=payload)
        return response.json()

    def publish_to_devto(self, article_data):
        """发布到Dev.to"""
        headers = {
            'api-key': self.config['devto_token'],
        }

        url = 'https://dev.to/api/articles'

        payload = {
            'article': {
                'title': article_data['title'],
                'body_markdown': article_data['content'],
                'published': True,
                'main_image': article_data.get('cover_image'),
                'tags': article_data.get('tags', []),
                'canonical_url': article_data['canonical_url']
            }
        }

        response = requests.post(url, headers=headers, json=payload)
        return response.json()

    def create_social_posts(self, article_data):
        """创建社交媒体帖子"""
        posts = {
            'twitter': self.create_twitter_post(article_data),
            'linkedin': self.create_linkedin_post(article_data),
            'reddit': self.create_reddit_post(article_data)
        }
        return posts

    def create_twitter_post(self, article_data):
        """创建Twitter帖子"""
        template = f"""
🚀 新文章发布：{article_data['title']}

{article_data['description']}

🔗 {article_data['url']}

#AI #Productivity #Career
        """.strip()
        return template

    def create_linkedin_post(self, article_data):
        """创建LinkedIn帖子"""
        template = f"""
我刚刚发布了一篇新文章关于{article_data['title']}。

在这篇文章中，我分享了：
• {article_data['key_point_1']}
• {article_data['key_point_2']}
• {article_data['key_point_3']}

完整文章：{article_data['url']}

#ProfessionalDevelopment #Productivity #{article_data['category']}
        """.strip()
        return template

# 使用示例
if __name__ == '__main__':
    publisher = ContentPublisher()

    article = {
        'title': 'Your Article Title',
        'content': 'Your article content in markdown...',
        'description': 'Brief description...',
        'url': 'https://yourblog.com/article',
        'canonical_url': 'https://yourblog.com/article',
        'tags': ['AI', 'Productivity', 'Career'],
        'cover_image': 'https://yourblog.com/image.jpg',
        'key_point_1': 'First key point',
        'key_point_2': 'Second key point',
        'key_point_3': 'Third key point',
        'category': 'AI'
    }

    # 发布到各平台
    medium_result = publisher.publish_to_medium(article)
    devto_result = publisher.publish_to_devto(article)

    # 生成社交媒体帖子
    social_posts = publisher.create_social_posts(article)

    print("Publishing complete!")
```
