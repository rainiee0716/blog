# 内容创作模板

## 深度分析文章模板

### 元数据
```yaml
---
title: '吸引人的标题 - 包含主要关键词'
description: '简洁的描述，120-150字符，包含关键词'
pubDate: 'YYYY-MM-DD'
heroImage: '../../../assets/hero-article-name.jpg'
category: 'AI|Career|Mindset|Productivity'
keywords: ['keyword1', 'keyword2', 'keyword3']
---
```

### 文章结构

#### 1. 引入 (200-300字)
- **问题引入**: 描述读者面临的挑战
- **数据支撑**: 引用相关研究或统计
- **价值承诺**: 说明文章将解决什么问题

#### 2. 背景/现状分析 (400-500字)
- **当前状况**: 描述行业现状
- **问题根源**: 分析问题产生原因
- **影响范围**: 说明问题的影响程度

#### 3. 核心概念/框架 (600-800字)
- **概念定义**: 清晰定义关键概念
- **框架介绍**: 详细介绍分析框架
- **图表说明**: 使用图表或表格说明

#### 4. 深度分析 (800-1000字)
- **多角度分析**: 从不同角度分析问题
- **案例研究**: 提供真实案例支撑
- **数据对比**: 使用数据对比说明观点

#### 5. 实施方案 (600-800字)
- **具体步骤**: 提供可操作的步骤
- **工具推荐**: 推荐相关工具和资源
- **注意事项**: 说明常见问题和解决方案

#### 6. 总结与展望 (200-300字)
- **关键要点**: 总结文章要点
- **行动号召**: 鼓励读者采取行动
- **后续预告**: 预告相关内容

### 检查清单
- [ ] 字数: 2500+
- [ ] 包含至少1个图表
- [ ] 3个以上案例研究
- [ ] 数据支撑观点
- [ ] 可操作建议
- [ ] 内部链接3-5个
- [ ] SEO优化

## 实用指南模板

### 文章结构

#### 1. 概述 (150-200字)
- **适用场景**: 说明指南适用情况
- **预期效果**: 说明学习后能达到的效果
- **所需时间**: 完成指南所需时间

#### 2. 准备工作 (200-300字)
- **工具准备**: 列出需要的工具
- **前提条件**: 说明必要的知识或技能
- **时间安排**: 建议的时间安排

#### 3. 步骤详解 (1000-1200字)
- **分步说明**: 每个步骤详细说明
- **截图说明**: 关键步骤配图
- **代码示例**: 如需要，提供代码示例
- **常见问题**: 每步骤的常见问题

#### 4. 进阶技巧 (300-400字)
- **优化建议**: 提供进阶优化建议
- **相关技巧**: 相关的其他技巧
- **资源推荐**: 进一步学习的资源

#### 5. 故障排除 (200-300字)
- **常见问题**: 列出常见问题
- **解决方案**: 提供解决方案
- **求助渠道**: 如何获得更多帮助

### 检查清单
- [ ] 字数: 1500-2000
- [ ] 步骤清晰可操作
- [ ] 包含截图或示例
- [ ] 故障排除完整
- [ ] 预期效果明确

## 行业观察模板

### 文章结构

#### 1. 新闻引入 (200-300字)
- **新闻概述**: 简要介绍新闻事件
- **背景说明**: 提供必要背景
- **重要性说明**: 为什么重要

#### 2. 深度分析 (600-800字)
- **影响分析**: 分析对不同群体的影响
- **趋势判断**: 判断长期趋势
- **专家观点**: 引用专家观点

#### 3. 实用建议 (400-500字)
- **应对策略**: 如何应对变化
- **机会把握**: 抓住什么机会
- **风险规避**: 如何规避风险

#### 4. 相关资源 (200-300字)
- **延伸阅读**: 推荐相关文章
- **工具资源**: 相关工具和资源
- **讨论社区**: 讨论和交流的地方

### 检查清单
- [ ] 字数: 1200-1500
- [ ] 时效性强
- [ ] 观点明确
- [ ] 建议实用
- [ ] 资源相关

## 个人经验模板

### 文章结构

#### 1. 背景介绍 (200-300字)
- **起点描述**: 描述起始情况
- **问题挑战**: 面临的挑战
- **动机说明**: 为什么做这件事

#### 2. 过程描述 (800-1000字)
- **分阶段描述**: 按阶段描述过程
- **关键决策**: 重要决策点和原因
- **困难挫折**: 遇到的困难和如何克服
- **转折点**: 关键转折点

#### 3. 结果展示 (400-500字)
- **具体数据**: 用数据展示结果
- **前后对比**: 对比前后变化
- **意外收获**: 意外的收获和发现

#### 4. 经验总结 (400-500字)
- **成功因素**: 成功的关键因素
- **失败教训**: 失败的教训
- **可复制性**: 哪些经验可以复制
- **适用范围**: 经验的适用范围

#### 5. 行动建议 (300-400字)
- **类似情况**: 类似情况如何应用
- **避免陷阱**: 需要避免的陷阱
- **资源推荐**: 相关资源推荐

### 检查清单
- [ ] 字数: 1500+
- [ ] 真实可信
- [ ] 数据支撑
- [ ] 经验可借鉴
- [ ] 诚实面对失败
